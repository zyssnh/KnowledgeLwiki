# eclipse

### Linux

#### CentOS-7

**安装目录解压**
```shell
# 指定路径
[user@hostname]$ tar -zxvf eclipse-inst-jre-linux64.tar.gz -C /path/../..
```

**打开**
```shell
[user@hostname]$ ./eclipse
```

**快捷方式**
```shell
[user@hostname]$ 
```


###### CentOS语言设置
```shell
[user@hostname]$ sudo vim /etc/locale.conf

```
